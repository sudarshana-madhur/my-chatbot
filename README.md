# My Chatbot

My Chatbot is a personal, individually-focused chatbot designed specifically for a single user. Powered by Gemini models (requires a Gemini API key), it is intended to serve as a private, personalized AI companion rather than a public-facing service.

## Features & Overview

- **Continuous Learning & Memory:** The chatbot slowly builds a memory and understanding of you. By continuously learning from your chats, it evolves over time and molds itself to grow closer to your personality and preferences.
- **Full-Stack Next.js:** Built entirely with Next.js, seamlessly integrating both a robust API backend and a responsive client-side interface.
- **Firebase App Hosting:** Securely hosted and managed using Google Firebase.
- **Progressive Web App (PWA):** Fully supports PWA capabilities, allowing you to install and use it as a standalone app on your devices.
